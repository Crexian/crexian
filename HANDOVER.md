# 🎮 Retro-Arcade Portfolio Handover Guide
> **안티그래비티(Antigravity) AI 개발 어시스턴트를 위한 프로젝트 연계 핸드오버 가이드**

이 문서는 다른 컴퓨터에서 안티그래비티를 실행했을 때 AI 어시스턴트가 프로젝트의 현 상태, 디자인 시스템의 상세 규칙, 그리고 파일 구조를 즉시 파악하고 이어서 개발을 원활히 진행할 수 있도록 안내하는 가이드라인입니다.

---

## 1. 프로젝트 개요 & 현 상태
본 프로젝트는 **90년대 고전 게임기 화면(NES/SNES/Game Boy) 및 JRPG 메뉴판** 스타일을 정밀하게 재현한 **디지털 미디어 & 크리에이티브 탐구 포트폴리오**입니다.

### 최근 완료된 주요 기능 (Phase 1 ~ 4)
*   **완전한 각진 픽셀 테마 (True Retro-Arcade)**: 둥근 모서리(`border-radius: 16px`)를 전면 폐기하고, 날카로운 직각(`border-radius: 0px`) 테두리와 입체감을 더하는 3D 인셋 베벨(Bevel) 그림자를 카드, 버튼, 모달창에 적용했습니다.
*   **통일된 픽셀 폰트 (Galmuri9)**: 너무 둥글둥글했던 폰트(`NeoDungGeunMo`) 대신 칼같이 곧고 날카로운 8비트 감성의 **`Galmuri9` (갈무리9)** 서체로 본문과 헤드라인을 통일했습니다. (영문은 `Pixelify Sans`, `Press Start 2P`, `VT323` 복합 사용)
*   **상단 LCD HUD Dashboard**: 오락실 화면 상단바처럼 실시간 점수(1UP Score, 스크롤/클릭 연동), 현재 페이지 위치(STAGE 01 ~ 04), 모은 동전 수(COINS 크레딧)를 보여주는 HUD 계기판을 실시간 JS 연동으로 구현했습니다.
*   **JRPG 퀘스트 로그 (성장 루프)**: 성장 단계를 세로형 퀘스트 목록으로 배치했습니다. 스크롤이나 호버 시 JRPG 특유의 **선택 손가락 포인터(👉)**가 통통 튕기며 활성화됩니다.
*   **인벤토리 퀵슬롯 (기록 보드)**: 5개의 카드형 보드를 게임 아이템 단축창(`SLOT 01~05`) 형태로 디자인했습니다.
*   **8비트 사운드 & 코인 물리 이스터에그**: Web Audio API로 신디사이징된 8비트 효과음(코인 소리, 파워업, 클릭, 호버)과 클릭 좌표에서 동전들이 튀어나와 튕기는 Canvas 2D 물리 엔진을 탑재했습니다.

## 2. 개발 및 실행 환경
*   **기술 스택**: 순수 정적 웹 (HTML5, Vanilla CSS3, Vanilla JS).
*   **빌드 도구**: 없음 (컴파일 또는 빌드 과정이 필요하지 않은 정적 파일 모음).
*   **실행 방법**:
    1.  레포지토리 클론 후, 임의의 로컬 정적 웹 서버를 통해 `index.html`을 실행합니다.
    2.  추천 도구: VS Code의 **Live Server** 확장 프로그램을 사용하여 포트 `5500` 등에서 원클릭 로컬 호스트 실행.
*   **배포 환경**: GitHub 푸시 시 **Vercel**을 통한 자동 CI/CD 트리거 및 호스팅 배포.

## 4. 디자인 약속 & 제약 사항 (중요)
새로운 작업을 진행하는 AI 어시스턴트는 다음 디자인 가이드를 반드시 엄수해야 합니다.
1.  **각진 픽셀 형태의 유지**: 어떠한 박스 요소도 둥글기(`border-radius`)를 가져서는 안 됩니다. 둥근 모서리가 발견되면 `border-radius: 0px !important;`로 오버라이드해야 합니다.
2.  **외곽선과 3D 섀도우**: 기본 테두리는 `3.5px solid var(--text)` (다크/라이트 모드 대응)를 사용하며, 텍스트 상자 등 입체 3D 효과가 필요한 요소는 `box-shadow: inset 3px 3px 0px rgba(255,255,255,0.25), inset -3px -3px 0px rgba(0,0,0,0.15), 6px 6px 0px var(--text);` 패턴을 유지합니다.
3.  **통일된 픽셀 폰트**: 모든 한국어 문구는 `Galmuri9` (갈무리9) 폰트를 적용해야 합니다. `Inter`나 일반 Sans-serif를 섞으면 디자인 통일성이 깨집니다.
4.  **캐시 방지(Cache Busting)**: CSS 파일 수정 시 `index.html` 내 `<link rel="stylesheet" href="styles.css?v=1.0.x">`의 버전 쿼리 파라미터를 점진적으로 증가시켜 브라우저 캐시 문제를 예방해야 합니다.

## 5. 안티그래비티 어시스턴트를 위한 다음 단계 추천
다음 컴퓨터에서 이어서 진행할 만한 추가 작업 아이디어 목록입니다:
-   **동전 누적 이스터에그 보상**: 누적 Coins가 일정 수준(예: 30개 또는 50개)을 넘어가면 숨겨진 미니 도트 애니메이션이나 오락실 트로피(숨겨진 업적 달성!) 팝업 메시지를 출력하는 피처 추가.
-   **음향 볼륨 제어 보완**: 볼륨을 완전히 끄는(Mute) 기능 외에, 8비트 사운드가 너무 튀지 않도록 오실레이터의 Gain 노드 볼륨 파라미터를 좀 더 미세하게 튜닝.
-   **반응형 레이아웃 정밀 검증**: 극단적인 모바일 모니터 가로 너비(320px 대)에서 상단 HUD 계기판과 JRPG 퀘스트 로그가 찌그러지지 않고 정상적인 직사각형 그리드를 유지하는지 CSS 미디어 쿼리 세부 조정.
