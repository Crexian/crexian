# 🎮 Blue Archive Style Modern SF Portfolio Handover Guide
> **안티그래비티(Antigravity) AI 개발 어시스턴트를 위한 프로젝트 연계 핸드오버 가이드**

이 문서는 다른 컴퓨터에서 안티그래비티를 실행했을 때 AI 어시스턴트가 프로젝트의 현 상태, 디자인 시스템의 상세 규칙, 그리고 파일 구조를 즉시 파악하고 이어서 개발을 원활히 진행할 수 있도록 안내하는 가이드라인입니다.

---

## 1. 프로젝트 개요 & 현 상태
본 프로젝트는 **[블루 아카이브] 청량하고 투명한 모던 SF 학원물 감성 (하늘색, 화이트, 글래스모피즘, 미니멀 라인, 똑- 물방울 효과음)** 스타일을 정밀하게 재현한 **디지털 미디어 & 크리에이티브 탐구 포트폴리오**입니다.

### 최근 완료된 주요 기능 (v3.1.0)
*   **블루 아카이브 테마 오버홀**: 따뜻한 베이지 양피지 톤에서 청량하고 맑은 아쿠아 블루/화이트 그라데이션(`#74c0fc` ~ `#e7f5ff` ~ `#ffffff`)으로 전면 교체되었습니다.
*   **미니멀 라인 & 소프트 섀도우**: 기존의 투박한 검은색 3.5px solid 테두리와 어두운 3D 섀도우를 전면 배제하고, `1px solid var(--line)`의 아주 연한 하늘색/화이트 보더와 부드러운 소프트 섀도우를 결합했습니다.
*   **글래스모피즘 (Glassmorphism)**: 카드 및 모달 창의 배경에 `backdrop-filter: blur(16px)`를 도입하여 청량하고 투명한 SF 분위기의 홀로그램 질감을 연출했습니다.
*   **아바타 헤일로 (Halo) 애니메이션**: 캐릭터 상태창 아바타에 블루 아카이브의 핵심 기호인 회전하는 원형 점선 후광(Halo) 데코레이션을 탑재했습니다.
*   **맑은 하늘 웰컴 스크린**: 시작화면의 마비노기 모닥불 연출을 폐기하고, 푸른 하늘 배경과 `SCHALE EXPLORER` 로고, `TAP TO CONNECT` 점멸 버튼을 탑재했습니다.
*   **Web Audio API 물방울 효과음**: 버튼/카드 클릭 시 맑게 튀는 "똑-" 소리(Pitch sweep) 및 디지털 차임 아르페지오 사운드를 직접 신디사이징하여 제공합니다.
*   **모바일 모달 스크롤 완벽 해결**: 모바일 환경에서 모달 기획서 등 내부 콘텐츠가 잘리지 않도록 `.modal-card` 높이를 `92vh`로 확보하고, 내부 `.modal-body`에 flex-scroll 및 `-webkit-overflow-scrolling: touch`를 설정하여 스크롤 문제를 근본적으로 해결했습니다.
*   **자바스크립트 기반 모바일 스크롤 락**: 모달이 열려 있을 때 뒷배경 바디가 흘러내리지 않도록 `position: fixed` 상태로 고정하고, 닫힐 때 원래 위치로 정확히 복구하는 스크롤 락 스크립트가 구현되어 있습니다.

## 2. 개발 및 실행 환경
*   **기술 스택**: 순수 정적 웹 (HTML5, Vanilla CSS3, Vanilla JS).
*   **빌드 도구**: 없음 (컴파일 또는 빌드 과정이 필요하지 않은 정적 파일 모음).
*   **배포 환경**: GitHub 푸시 시 **Vercel**을 통한 자동 CI/CD 트리거 및 호스팅 배포.

*   [`index.html`](file:///d:/pot/index.html): 전체 페이지 구조. 상단 HUD, 캐릭터 정보창, 인벤토리 카드 그리드, 성장 루프 퀘스트 로그, 캠페인 제안서 탭 모달 등이 포함되어 있습니다.

## 4. 디자인 약속 & 제약 사항 (중요)
새로운 작업을 진행하는 AI 어시스턴트는 다음 디자인 가이드를 반드시 엄수해야 합니다.
1.  **청량하고 투명한 톤앤매너 유지**: 투박하고 어두운 블랙 테두리나 브라운/베이지 등 따뜻한 양피지 색감의 흔적을 섞어서는 안 됩니다. 아쿠아 블루, 클린 화이트, 소프트 실버 블루 톤을 고수하십시오.
2.  **글래스모피즘 가이드**: 투명 질감이 필요한 카드 요소는 `backdrop-filter: blur(16px)`와 반투명 배경색(`rgba(255, 255, 255, 0.45)`)을 결합하여 표현합니다.
3.  **세련된 산세리프 폰트**: 타이틀에는 우아한 `Cinzel`을 제한적으로 사용하되, 본문은 픽셀 도트 글꼴이 아닌 현대적이고 깔끔한 `Pretendard`, `Inter` 혹은 `Noto Sans KR` 서체를 조화롭게 매칭해야 합니다.
4.  **캐시 방지(Cache Busting)**: CSS 또는 JS 파일 수정 시 `index.html` 내 로드 링크에 붙은 버전 쿼리 파라미터(`?v=3.1.x`)를 하나씩 점진적으로 올려 브라우저 캐시 문제를 방지해야 합니다.

## 5. 안티그래비티 어시스턴트를 위한 다음 단계 추천
*   **동전 물리 이스터에그 보강**: `TAP TO CONNECT` 단계 및 로비 클릭 시 뿜어져 나오는 동전들의 스킨을 블루 아카이브 스타일의 크리스탈이나 별 조각 느낌으로 가볍게 리스킨하여 비주얼 일관성을 높이는 방안.
*   **반응형 레이아웃 극단 검증**: 모바일 환경에서 사용자가 기획서 모달 내부 탭을 조작하거나 스태프 롤을 볼 때, 텍스트 가독성이 명확하고 오버플로우가 생기지 않는지 기기별로 꼼꼼히 점검.
